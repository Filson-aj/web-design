# 📘 MODERN WEB DEVELOPMENT 

## Course Title
Learning Web development the modern way

## Course Level
Beginner to Intermediate

## Duration
12 Weeks (3 Months)

## Delivery Mode
Theory + Practical (Hands-on Coding Sessions)

---

## 📖 Course Description

This course introduces students to the fundamentals of web design. It focuses on building strong foundational skills required to design and develop modern, responsive websites.

Students will learn how to structure web pages using HTML and style them using CSS, including layout design, responsiveness, and basic animations. The course emphasizes practical learning through exercises and real-world projects.

---

## 🎯 Course Objectives

By the end of this course, students will be able to:

- Understand how websites are built and structured
- Write clean and semantic HTML code
- Style web pages using CSS
- Build responsive web layouts
- Use Flexbox and Grid for layout design
- Develop simple to intermediate web pages and websites

---

## 🧭 COURSE OUTLINE (WEEKLY BREAKDOWN)

### Week 1: Introduction to Web Design
- Introduction to web design and development
- How websites work
- Overview of HTML and CSS
- Setting up development environment (VS Code)

**Practical:** Create first HTML page

---

### Week 2: HTML Basics I
- HTML structure
- Headings, paragraphs, and text formatting
- Comments in HTML

**Practical:** Personal bio page

---

### Week 3: HTML Basics II
- Lists (ordered and unordered)
- Links and navigation
- Images and file paths

**Practical:** Simple multi-page website

---

### Week 4: Semantic HTML & Forms
- Semantic tags (header, section, footer, nav)
- Forms and input elements
- Labels and buttons

**Practical:** Contact form page

---

### Week 5: HTML Tables & Multimedia
- Tables (rows and columns)
- Audio and video embedding
- YouTube embedding

**Practical:** Student result table page

---

### Week 6: Introduction to CSS
- What is CSS?
- Types of CSS (inline, internal, external)
- CSS syntax and selectors

**Practical:** Style previous HTML pages

---

### Week 7: CSS Styling Fundamentals
- Colors (HEX, RGB, named colors)
- Fonts and typography
- Text alignment and spacing
- Box model (margin, padding, border)

**Practical:** Styled profile card

---

### Week 8: CSS Layout Basics
- Display properties
- Positioning (relative, absolute, fixed)
- Introduction to Flexbox

**Practical:** Navigation bar layout

---

### Week 9: Flexbox & Grid
- Flexbox alignment and spacing
- CSS Grid basics
- Layout building techniques

**Practical:** Website homepage layout

---

### Week 10: Responsive Web Design
- Mobile-first design
- Media queries
- Responsive units (%, rem, vh, vw)

**Practical:** Mobile-friendly website conversion

---

### Week 11: Advanced CSS
- Transitions and animations
- Hover effects
- Pseudo-classes and pseudo-elements
- CSS variables

**Practical:** Animated landing page

---

### Week 12: Final Project
- Full website development
- Portfolio or business website
- Project presentation
